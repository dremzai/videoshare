# videoshare
微信短视频推广系统

微信短视频推广系统前端源码 
app小程序按照推广效果付费 
以短视频为主体,点赞,分享等效果付费

商家可以新增主题活动,管理相关的视频,以及设置规则,充值到主题中
用户可以根据分享的视频情况直接提现



目前前端项目已开源 欢迎使用
